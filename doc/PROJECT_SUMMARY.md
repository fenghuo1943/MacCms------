## 🎯 关键特性详解

### 1. Token Bucket 限流器

```python
class TokenBucket:
    - rate: 每秒产生的令牌数
    - capacity: 桶容量（最大突发）
    - acquire(): 获取令牌（阻塞/非阻塞）
```

**优势：**
- 平滑控制请求速率
- 允许适度突发
- 线程安全

### 2. 速率监控器

```python
class RateLimitMonitor:
    - record_request(): 记录请求结果
    - get_current_rate(): 获取当前速率
    - get_stats(): 获取统计数据
```

**功能：**
- 实时速率监测
- 成功率统计
- 限流次数追踪

### 3. 断点续传

```python
# 自动保存到 fetch_stats.json
{
    "start_time": "2024-01-15T10:00:00",
    "total_processed": 5000,
    "total_success": 4500
}
```

**机制：**
- 每处理一个视频就保存
- 下次启动自动加载
- 基于数据库状态判断

### 4. 智能匹配

```python
匹配策略：
1. 名称模糊匹配（包含关系）
2. 年份精确匹配
3. 唯一性验证（只接受单个匹配）
```

**状态标记：**
- 0: 成功
- 1: 未处理
- 2: 多个结果
- 3: 无结果
- 4: 错误
